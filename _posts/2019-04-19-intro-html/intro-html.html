はじめに

政治学においてウェブサイトの知識が必要になる局面としては少なくとも次の2つがあります。

  1. ウェブスクレイピング
  2. 個人ウェブサイトの作成

そこで、今回はこれらを実施する際に必要となる最低限の知識について紹介します。

基本的に、ウェブサイトは.htmlファイルによって構成されています。 .htmlファイルには文章や画像のパスなどのコンテンツが記入されています。

HTMLはマークアップ言語でタグ付けすることにより各要素の役割と外見を紐付けます。 具体的には.cssファイルによってそれぞれのタグなどをどのように表示するか指定します。

その他に、JavaScriptによってより動的な表現・サービスを行ったり、PHPによってサイトの構築と動的サービスを一体的に行ったりします。 ひとまず、ウェブスクレイピングには.htmlファイルの、ウェブサイト作成にはそれに加えて.cssファイルの知識があれば最低限のことはできます。

HTML CSS JS PHP
ウェブスクレイピング 必要 必要ない 必要ない 必要ない
ウェブサイト作成 必要 必要 もしかすると必要 もしかすると必要

HTML

まず、Bootstrap 4というフレームワークのサンプル.htmlファイルを見てみましょう。

フレームワークとはある言語を実行する際の便利なテンプレートのようなものです。

これはブラウザ上ではこのように表示されます。

ブラウザとはウェブサイトを閲覧するためのソフトでFirefoxやGoogle Chromeなどがあります（Internat ExplorerやEdgeというものはFirefoxとGoogle Chromeをダウンロードするためのソフトです）。

以下ではHTMLの代表的なタグと要素について説明していきます。

headとbody

なんとなく分かると思うのですが、基本的に<...>がタグの始まりで</...>がタグの終わりになっています。 そして、...がタグの名前になっています。

.htmlファイルではまずheadとbodyに大きく分けられます。 headではメタデータの設定（例えば文字コードやサイトのタイトル）や.cssファイルの読み込みなどを行います。 ときどき、この例のようにbody内でJavaScriptを読み込むときもあります。

次に、body内においてウェブサイト上に表示される情報を記述します。 ウェブスクレイピングをする際には基本的にbody内を見れば事足ります。

なお、<!--と-->で囲まれている箇所はコメントアウトされます。

hタグ

hタグとは論文や本における見出しのようなものです。 上記の例で言えば、<h1>Hello, world!</h1>がそれに当たります。

h2、h3と番号が大きくなるにつれてより小さな見出しになっていきます。 例えば、このページのhタグ次のようになっています。


<h1>政治学のためのHTML/CSS入門</h1>

<h2 id="Intro">はじめに</h2>

<h2 id="HTML">HTML</h2>

<h3 id="headbody">headとbody</h3>

...

pタグとaタグ

さて、このページのHTMLの節の最初の一文は.htmlファイルでは次のように記述されます。


<p>まず、<a href="https://getbootstrap.com/">Bootstrap 4</a>というフレームワークのサンプル<code>.html</code>ファイルを見てみましょう。</p>

pタグは文章を記述する箇所で、aタグはリンクを張るタグになります。 aタグの中のhref =の箇所でリンク先のパスやURLを記述します。

リスト

このページの冒頭で番号付きの箇条書きの箇所がありますが、これは以下の様に記述されます。


<ol type="1">
  <li>ウェブスクレイピング</li>
  <li>個人ウェブサイトの作成</li>
</ol>

olタグは順序付きリストを作成するタグで、liタグはリストの項目を記述します。 ちなみに、ulにすると順序なしリストになります。

  • ウェブスクレイピング
  • 個人ウェブサイトの作成

<ul>
  <li>ウェブスクレイピング</li>
  <li>個人ウェブサイトの作成</li>
</ul>

まず、tableタグでテーブル全体を囲み、theadタグとtbodyタグでヘッダーとボディを囲みます。 そして、trタグで各行を決めて、tdタグで各セルの中身を記述しています。

divタグ

divタグとは分割を意味するタグです。 このページのタイトルは以下のようになっています。


<div class="d-title">
  <h1>政治学のためのHTML/CSS入門</h1>
  <p>スクレイピングと個人ウェブサイト作成のために必要なHTMLとCSSの基礎知識を説明します。</p>
</div>
  • divタグそれ自体に意味はないですが、そこからどこまでが一つの塊なのかを明らかにするのに役立ちます。

idとclass

タグの中でidやclassを指定することができます。 例えば、<h2 id="Intro">はじめに</h2>や<tr class="header">...</tr>といった形です。

これらはそれぞれ特定のタグの目印となるものですが、以下のような違いがあります。

  • idは複数のタグで同じものを使えない。
  • classは複数のタグで共通のものが使える。
しかし、そこまで厳密に使い分けているとも限りません。

CSS

続いて、HTMLとCSSの編集方法を簡単に学びます。 なお、実際にウェブサイトを作成する際にはR Markdownで作る、Bootstrap 4などのフレームワークを使うなどしたほうがいいでしょう。

練習用の.htmlファイルと.cssファイルを圧縮したzipファイルをダウンロードして、適当な場所で解凍してください。 中に、index.htmlとsample.cssというのがあるはずなので、それらを適当なテキストエディタで開きます。

ガチ勢はVimやEmacs、宗教論争に巻き込まれたくないライト層はAtomやSublime Textを使うのが定番でしょうか。 そんなのどうでもいいエディタにこだわりのない人はRStudioでもできます。

なお、編集結果はブラウザで見ながら作業をしたいと思うので、index.htmlはブラウザでも開いてください。

index.htmlにはある程度記述がありますが、sample.cssは空白のはずです。 以下ではこれらを編集しながら使い方を学びます。

index.htmlのheadタグ内に


<link rel="stylesheet" href="sample.css">

という記述があります。 これは、（index.htmlと同じディレクトリにある）sample.cssをスタイルシートとして読み込む、という意味です。

リンクを貼る

このサンプルデータは2019年の大阪市長および大阪府知事のダブル選挙の結果のページになっています。 もとのデータは選挙ドットコムから拝借しました。

そうならば、引用元のページのリンクを貼るというのがマナーでしょう。 前述したようにリンクを貼るにはaタグを使います。 例えば、


<p>以下の情報は選挙ドットコムからお借りしました。</p>

を


<p>以下の情報は<a href="https://go2senkyo.com/">選挙ドットコム</a>からお借りしました。</p>

と書き換えてください。

そして、sample.thmlをエディタ上で保存し（Ctrl + Sを押し）、ブラウザ上で更新して（Ctrl + Rを押して）ください。 すると、「選挙ドットコム」の場所にリンクが貼られているはずです。

このCtrl + SとCtrl + Rはウェブサイトの作成において必須です。

ちなみに、以下のようにtarget="_blank"とすることでリンク先を新しいタブで開きます。


<p>以下の情報は<a href="https://go2senkyo.com/" target="_blank">選挙ドットコム</a>からお借りしました。</p>

全体への適用

ここからはsample.cssを編集して文字の見た目を変化させていきたいと思います。

まずは、デフォルトのダサいフォントを変えたいと思います。 何がきれいなフォンとかは人それぞれだとは思いますが、僕はGoogleとAdobeのNoto Sansが好きです。

また、（読み込みに時間がかかるという問題はあるものの）ウェブ上で無料でフォントをインポートできるので、デバイスに依存せずにフォントを決めることができます。 sample.cssの冒頭で、


@import url('https://fonts.googleapis.com/css?family=Noto+Sans+JP');

と記述することでnoto Sans JPをオンラインでインポートします。

そして、次のように記述すると全ての文字にNoto Sans JPが適用されます。


* {
  font-family: 'Noto Sans JP';
}

sample.cssを保存し、ブラウザを更新すると（分かりにくいですが）フォントが少しスマートになったような気がするはずです。

さて、CSSの記述ですが、まず最初にどのタグに適用させるかを決めます。 今回は*とすることで全てのタグに適用させています。

続いて、{}内に適用させる内容を記述します。 今回はfontfamilyでフォントの種類を指定しています。 そして、:の後ろに指定内容を記述し、;で閉じます。

特定のタグへの適用

特定のタグに適用したい場合も{}の前にタグの名前を記述します。 例えば、h1タグを中央揃えにしたい場合は以下のように記述します。


h1 {
  text-align: center;
}

複数のタグを一括して選択することもできます。 例えば、h1からh3タグ全ての文字の太さを標準にします。


h1, h2, h3 {
  font-weight: normal;
}

同じタグであれば一括して指定されます。 例えば、


td {
  padding-left: 15px;
}

とすると、表の全てのセルの左側に15ピクセル分の余白が生まれます。

CSSパス

特定の条件のタグだけに適用させたい場合もあります。 例えば、候補者の情報だけを右揃えにしたい場合、tdタグに一括で適用させるとうまく行きません。 そのような場合は、次のように書くことでtbodyタグ内のtdタグにだけ適用させることができます。


tbody td {
  text-align: right;
}

CSSセレクタ

より細かく指定したい場合はidやclassといったCSSセレクタを使用します。 例えば、大阪市長選挙の文字を緑、大阪府知事前挙の文字を青色に変えたい場合は、次のように書きます。

本当はRGB方式で色指定をしたほうがいいです。 

#city {
  color: green;
}

#prefecture {
  color: blue;
}
  • #は特定のidがついているタグだけに適用するという意味です。

続いて、以下のように書くことで当選者の背景を赤色にすることができます。


.elected {
  background-color: red;
}
  • 特定のclassに適用したい場合は.を使います。

レスポンシブデザイン

レスポンシブデザインとはディスプレイの大きさによって表示を変化させるようなウェブデザインのことです。 これによPCで開いてもスマートフォンで開いても適切に表示されるようになります。

Bootstrapではグリッドシステムによってこれを実現しているのですが、そのために、まずはcontainerクラスのdivタグ内に内容を記述する必要があります。


<h1>Hello, world!</h1>

を


<div class="container">
  <h1>Hello, world!</h1>
</div>

のように書き換えます。 すると、若干左に余白が生まれてナイスな感じになります。

コンポーネント

.htmlの中身についてはドキュメントのコンテンツやコンポーネント、ユーティリティから実装したいものを見つけ、サンプルコードをコピー&ペーストして修正するという形になります。

例えば、タイトルをいい感じにしたいのでJumbotronを見てみると、以下のようなサンプルコードがあります。


<div class="jumbotron">
  <h1 class="display-4">Hello, world!</h1>
  <p class="lead">This is a simple hero unit, a simple jumbotron-style component for calling extra attention to featured content or information.</p>
  <hr class="my-4">
  <p>It uses utility classes for typography and spacing to space content out within the larger container.</p>
