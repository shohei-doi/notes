はじめに

ある変数がどのように分布しているのかを示すための図を作ります。 具体的にはヒストグラム、カーネル密度、棒グラフについて説明します。

基本的に

  • 標準作図関数
  • tidyverseのggplot2
  • インタラクティブな図を作成するplotly

による方法を説明します。

一番使うのはggplot2なので、詳しくなります。

標準作図関数

Rで標準に備わっている作図関数を使います。

ヒストグラム

分布を示す代表的な図はヒストグラムです。 貿易額tradeのヒストグラムを作図します。


hist(data$trade)

左に歪んでいる場合は対数を取ると正規分布っぽくなります。


hist(log(data$trade))

カーネル密度

カーネル密度というヒストグラムを平滑化したグラフもあります。 ただし、欠損値を取り除き、density()で密度を計算してplot()で線グラフを作成します。


plot(density(log(data[!is.na(data$trade),]$trade)))

棒グラフ

離散変数の場合はヒストグラムではなく棒グラフを使います。 民主主義度democの棒グラフを作成します。 table()で度数分布表を作成し、barplot()で作図します。


barplot(table(data$democ))

ggplot2

tidyverseの中のggplot2というパッケージを使って作図します。 ggplot2では

  1. ggplot()という関数でオブジェクトを作成し、
  2. +で適当なグラフを重ね合わせていき、
  3. 適宜、aes()で使用する変数を指定する、

という流れになります。

%>%と+を間違えることがあります。

ヒストグラム

tradeのヒストグラムを作成します。


data %>% 
  ggplot() + 
  geom_histogram(aes(x = trade))

`stat_bin()` using `bins = 30`. Pick better value with `binwidth`.

対数を取る場合は直接対数を取るかすケースを変更する関数を重ねます。


data %>% 
  ggplot() + 
  geom_histogram(aes(x = trade)) + 
  scale_x_log10()

`stat_bin()` using `bins = 30`. Pick better value with `binwidth`.

Warning: Removed 3728 rows containing non-finite values (stat_bin).

カーネル密度


data %>% 
  ggplot() + 
  geom_density(aes(x = trade)) + 
  scale_x_log10()

Warning: Removed 3728 rows containing non-finite values (stat_density).

棒グラフ


data %>% 
  ggplot() + 
  geom_bar(aes(x = democ))

plotly

plotlyはインタラクティブな作図をするPlotlyを使うパッケージです。

ヒストグラム

変数を指定するときには~をつけます。


data %>% 
  plot_ly(x = ~trade, type = "histogram")

Warning: Ignoring 3728 observations

data %>% 
  plot_ly(x = ~log(trade), type = "histogram")

Warning: Ignoring 3728 observations

plotlyではggplot2のオブジェクトをggployly()で変換することができます。

パイプで繋ぐことはできないようです。 

p <- data %>% 
  ggplot() + 
  geom_histogram(aes(x = trade)) + 
  scale_x_log10()
ggplotly(p)

`stat_bin()` using `bins = 30`. Pick better value with `binwidth`.

Warning: Removed 3728 rows containing non-finite values (stat_bin).

棒グラフ

plotlyの場合は離散変数でもヒストグラムをちゃんと出力してくれます。


data %>% 
  plot_ly(x = ~democ, type = "histogram")
