はじめに

ある変数がどのように分布しているのかを示すための図を作ります。 具体的には散布図、箱ひげ図、バイオリンプロットについて説明します。

基本的に

  • 標準作図関数
  • tidyverseのggplot2
  • インタラクティブな図を作成するplotly

による方法を説明します。

一番使うのはggplot2なので、詳しくなります。

標準作図関数

Rで標準に備わっている作図関数を使います。

散布図

散布図を作成するにはplot()を使います。 ここでは国力cincと理想点idealの散布図を作成します。


plot(log(data$cinc), data$ideal)

一方が離散変数の場合、散布図は見にくくなります。 国力の代わりに民主主義度democを使ってみます。


plot(data$democ, data$ideal)

このようなときはjitter()で散らばるようにします。


plot(jitter(data$democ), data$ideal)

箱ひげ図

あるいは箱ひげ図というものを使う方法もあります。 boxplot()内でformulaを定義して使います。


boxplot(ideal ~ democ, data = data)

ggplot2

tidyverseの中のggplot2というパッケージを使って作図します。 ggplot2では

  1. ggplot()という関数でオブジェクトを作成し、
  2. +で適当なグラフを重ね合わせていき、
  3. 適宜、aes()で使用する変数を指定する、

という流れになります。

%>%と+を間違えることがあります。

散布図


data %>% 
  ggplot() + 
  geom_point(aes(x = cinc, y = ideal))+
  scale_x_log10()

Warning: Removed 7319 rows containing missing values (geom_point).


data %>% 
  ggplot() + 
  geom_jitter(aes(x = democ, y = ideal))

Warning: Removed 7999 rows containing missing values (geom_point).

箱ひげ図

箱ひげ図の場合、数値データだと分類してくれないのでas.factor()でカテゴリカルデータに変換します。


data %>% 
  ggplot() + 
  geom_boxplot(aes(x = as.factor(democ), y = ideal))

Warning: Removed 6740 rows containing non-finite values (stat_boxplot).

バイオリンプロット

箱ひげ図に似たものとしてバイオリンプロットというものがあります。


data %>% 
  ggplot() + 
  geom_violin(aes(x = as.factor(democ), y = ideal))

Warning: Removed 6740 rows containing non-finite values (stat_ydensity).

これらを重ね合わせることもできます。


data %>% 
  ggplot() + 
  geom_violin(aes(x = as.factor(democ), y = ideal)) + 
  geom_boxplot(aes(x = as.factor(democ), y = ideal))

Warning: Removed 6740 rows containing non-finite values (stat_ydensity).

Warning: Removed 6740 rows containing non-finite values (stat_boxplot).
